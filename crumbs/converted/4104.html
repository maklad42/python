4104

Cause

Paper size or media type that cannot be fed from cassette is specified.

What to Do

Press the printer 's Stop button to cancel printing.

In the paper settings specified when printing, change Page Size to A4, Letter, A5, B5, or Executive, and Media Type to plain paper. Then retry printing.

When using a custom paper size, set Page Size to Custom Paper Size and specify vertical and horizontal dimensions of the paper.

  • Setting Paper Dimensions (Custom Size)

Note

  • If you will not change the paper settings specified when printing, select the rear tray as the paper source.

  • Register the paper information to the printer correctly.
